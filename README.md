# Barber Web Application

Angular web application for barbers to manage their business, appointments, and customers.

## Features

- Barber authentication (register/login)
- Dashboard with earnings and stats
- Appointment management
- Customer history tracking
- AI recommendations
- Chat with customers
- Responsive design

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update `src/environments/environment.ts` with your API URL

3. Run development server:
```bash
npm start
```

4. Open [http://localhost:4200](http://localhost:4200)

## Tech Stack

- Angular 17
- TypeScript
- Tailwind CSS
- RxJS
- Axios (via HttpClient)
